# 📊 Sales Dataset Dashboard in Excel

## Overview

This project demonstrates the complete data analysis workflow in Microsoft Excel, from cleaning raw sales data to building an interactive dashboard. The dashboard provides key business insights through KPIs, charts, and slicers, making it easy to analyze sales performance.

## Dataset

* **Original Rows:** 113,038
* **Rows After Data Cleaning:** 112,026

## Project Workflow

1. Imported the sales dataset into Excel.
2. Cleaned the data by removing unnecessary records and improving data quality.
3. Checked for missing values and inconsistencies.
4. Created Pivot Tables for data analysis.
5. Built an interactive dashboard using charts, KPIs, and slicers.

## Dashboard Features

* 📈 Total Sales
* 💰 Total Profit
* 📦 Total Orders
* 🌍 Region-wise Sales Analysis
* 📅 Time-based Sales Trends
* 🛍️ Category & Product Performance
* 🎛️ Interactive Slicers for dynamic filtering
